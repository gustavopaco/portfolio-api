Projeto API para Portfólio de Desenvolvedor Web Full Stack.
======================================================
## Descrição do Projeto
<p align="justify">Projeto desenvolvido como API para o portfólio de desenvolvedor web full stack, com o objetivo de demonstrar as habilidades em desenvolvimento web, utilizando as tecnologias Angular, Spring Boot, HTML5, CSS3, Typescript, JavaScript, Bootstrap e Postgres.</p>
